# 🚀 Modern Portfolio - Zakir Raza

A professional, modern, and fully responsive portfolio website built with **React + Vite**, featuring dark/light mode, smooth animations, and optimized performance.

## ✨ Features

- ✅ **Dark/Light Mode** - Theme toggle with localStorage
- ✅ **Modern Design** - Neon green & purple gradient theme
- ✅ **Smooth Animations** - Framer Motion & AOS
- ✅ **Fully Responsive** - Works on all devices
- ✅ **SEO Optimized** - Meta tags and semantic HTML
- ✅ **Fast Performance** - Vite build tool
- ✅ **Contact Form** - Integrated with Web3Forms API

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

Visit `http://localhost:5173` to view the portfolio.

## 📁 Structure

```
src/
├── components/     # React components
├── data/          # Portfolio content
├── styles/        # CSS files
└── App.jsx        # Main app
```

## 🎨 Customization

Edit `src/data/portfolioData.js` to update:
- Personal info
- Education
- Skills
- Experience
- Projects
- Certificates

## 🛠️ Tech Stack

- React 19
- Vite
- Framer Motion
- React Icons
- React Scroll
- AOS (Animate On Scroll)
